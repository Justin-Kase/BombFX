# Bomb FX Presets

14 factory presets covering everything from subtle enhancement to wild experimental effects.

## Reverb Presets
- **Cathedral** — Large, spacious hall reverb with bright character
- **Small Room** — Tight, natural room ambience
- **Vocal Plate** — Classic plate reverb for vocals and instruments

## Delay Presets
- **Slapback** — Short delay for rockabilly/vintage vibe
- **Dub Echo** — Long feedback delay for reggae/dub
- **Ping Pong Madness** — Extreme stereo feedback delay

## Chorus Presets
- **Lush Chorus** — Rich modulation for synths and guitars
- **Subtle Shimmer** — Gentle chorus for adding width
- **Studio Thickener** — Subtle enhancement for mixing

## Combo Presets
- **Ambient Space** — Reverb + delay + chorus for atmospheric textures
- **Dream Machine** — Max everything for ethereal soundscapes
- **Psychedelic** — Fast chorus + short delay for wild modulation
- **Infinite Void** — Massive reverb + long delay for drone/ambient
- **Tight & Punchy** — Subtle everything for adding depth without washing out

## Usage

Load presets directly in your DAW's plugin browser, or manually load the XML files via the plugin's preset menu (if implemented).
